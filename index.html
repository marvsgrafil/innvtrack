FAQs

How do I register my class?

Teachers can sign up and input class details in the dashboard to get started.

Do students need accounts?

Not necessarily — each student only needs their QR code for scanning.

Can it be used offline?

Yes, it can temporarily store attendance locally and sync later when online.
